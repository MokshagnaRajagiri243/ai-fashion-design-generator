# AI Fashion Design Generator

## Problem Statement
Many students want to explore fashion design but lack artistic or technical skills. Traditional tools are expensive and time-consuming. This project uses AI to generate fashion designs from text prompts.

## Features
- AI Fashion Image Generation
- Style Selection
- Color Selection
- Occasion Selection
- Responsive UI

## Live Demo
https://dream-design-muse.lovable.app

## Tech Stack
- Lovable
- React
- AI Image Generation

## How to Use
1. Open the live demo.
2. Enter a fashion prompt.
3. Select style, color and occasion.
4. Click Generate Design.
5. View the generated fashion image.
